# ckau-to-es-de
This set of scripts is designed to automatically convert theme collections from Ckau (made for Batocera and Retrobat) for use in ES-DE.

# 🎮 Адаптация коллекций оформлений под ES-DE

Инструмент для автоматической адаптации коллекций оформлений Ckau (Batocera / Retrobat) под структуру **ES-DE**.

Скрипты копируют ROM и медиа, при необходимости оптимизируют изображения и видео, удаляют медиа-суффиксы и приводят файлы к формату, ожидаемому ES-DE.

---

## ✨ Возможности

* 📂 Копирование файлов (включая подпапки) в структуру ES-DE
* 🖼 (Опционально) Оптимизация изображений
* 🎬 (Опционально) Оптимизация видео
* 🧹 Удаление медиа-суффиксов (`-boxart`, `-fanart`, `-video` и т.д.)
* 🏷 Переименование медиа под имена ROM
* 📁 Раскладка медиа по подпапкам ROM
* 📋 Лог пропущенных файлов

---

## 📋 Требования

* Windows 8.1 / 10 / 11
* Windows 7 — требуется PowerShell ≥ 3.0 (рекомендуется 5.1)

---

## 📦 Состав архива

| Файл                   | Назначение                    |
| ---------------------- | ----------------------------- |
| `_Ckau-to-ES-DE.cmd`   | основной управляющий скрипт   |
| `_MoveRenameMedia.ps1` | умная обработка медиа         |
| `_ffmpeg.exe`          | обработка изображений и видео |

---

## 🚀 Быстрый старт

### 1️⃣ Подготовка

Поместите файлы:

```
_Ckau-to-ES-DE.cmd
_ffmpeg.exe
_MoveRenameMedia.ps1
```

в папку системы (например в `megadrive`).

---

### 2️⃣ Запуск

Запустите:

```
_Ckau-to-ES-DE.cmd
```

Скрипт:

* создаст папку `ES-DE`
* начнёт копирование файлов
* предложит оптимизацию медиа

---

### 3️⃣ Ответьте на вопросы

Во время работы появятся два вопроса:

* Уменьшить разрешение изображений?
* Уменьшить разрешение и размер видео?

**Y = Да**
**N = Нет**

⏱ Время на ответ — 60 секунд.
Если ничего не выбрано — по умолчанию выбирается **Да**.

#### Как изменить выбор по умолчанию

Найдите в cmd-скрипте строку:

```bat
choice /C YN /N /T 60 /D Y /M "Ваш выбор [Y/N]: "
```

и замените:

```
/D Y → /D N
```

---

## 🧠 Как работает сопоставление медиа

Скрипт пытается «умно» сопоставить медиа с ROM:

1. Точное совпадение имени
2. Совпадение после очистки тегов (`[ ]`, `( )`)
3. Если ROM не найден — удаляется только медиа-суффикс

⚠️ При сильных различиях имён потребуется ручная проверка.

---

## 📄 Лог пропущенных файлов

После выполнения создаётся файл:

```
_skipped_files_<system>.txt
```

В нём перечислены медиа, которые не удалось сопоставить.

**Можно игнорировать:**

* медиа для папок (Hack, New, Unl и т.д.)
* системные фанарты

---

## ⚠️ Важные замечания

### Разные имена систем

Названия систем в Batocera / Retrobat и ES-DE могут отличаться.

👉 Проверяйте документацию ES-DE и при необходимости переименовывайте папку системы.

---

### Одно медиа на несколько ROM

В Batocera / Retrobat одно и то же медиа может использоваться для нескольких ROM через gamelist.xml.

В ES-DE такой механики нет.

👉 При необходимости придётся продумать собственное решение.
